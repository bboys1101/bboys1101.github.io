2016-11-16

Why I Talk About TextIndent

Recently, I’ve found I wasn’t paying enough attention to semantic html for my previous project, which made me feel guilty and embrassed. After all, I always think of myself as a great helper in digital territory for my clients.

Well, it’s never too late to start picking up and building my foundation, during this whole process of studying, there is one question stuck my mind,

How people deal with logo when they also want to put the text inside of the tag so search engine can parse?

I’ve browsed many websites, seem many different ways to build the logo, however, there is a specific way attracts my attention.


Where da hell is the text inside the <a> tag?
I couldn’t see it.



I kept asking myself, because the only one thing exist on the website is the logo image, that’s a goddamn magic.

Thanks to the inspector, I notice that there is a css attribute called text-indent which I don’t know what it is, and this is how we meet.

What is Text-indent

According to w3schools, the text-indent property specifies the indentation of the first line in a text-block. For example, if you want to make a layout looks like this



–Pic from The Negra

You will only need to set the value in number larger than 0,

So you can image that if you set the negative number, the first line will move toward to left.

How to use it for this case

The concept is setting a very large number, no matter it is larger than 0 or less than 0, just make it move with a long distance, so it will disappear from the screen, and then we use background-img to show the logo, in this we can meet the requirement from the search engine, so it can be parsed as a logo with clear text, meanwhile we can show the image on the website, what a great solution.
