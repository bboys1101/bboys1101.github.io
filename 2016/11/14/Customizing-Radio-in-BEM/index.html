Customizing Radio in BEM

2016-11-14

Customizing Radio

I never take it serious about the radio of input until recently, a landing page required me to customizing it, and then I found out there are some tips can help me to build a radio in a way I want.

Typical Looking

I am not sure what about other Devs, for me, the first thought came into my mind for customizing radio, is inspecting the css, and overwriting the style attributes, however, this will lead me to a tough situation that no one would ever want to face.

Trust me, you don’t want to fight with this directly.

So how can we achieve it?

Basically, there are three features that can help us to achieve it.

  1. element+element selector in CSS.

  2. input[type=”radio”]:not(:checked),
    input[type=”radio”]:checked{
    position: absolute;
    left:-9999px;
    }

//unactive


//active


&__radio:checked + label {
background: $yellow;
color:#000;
}

&__label {
  padding: 24px 0px;
  text-align: center;
  flex: 1 30%;
  margin: 0px 10px;
  border: 1px solid;
  font-size: 1.8rem;
  cursor: pointer; 
  @include transition(all, 0.2s);

  &:hover {
    transform: scale(1.05);
    @include transition(all, 0.2s);
  }

}
```

Apparently, if the label is clicked, it triggers the radio to be checked, and then the label will have different background color.
